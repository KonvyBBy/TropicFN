# Konvy Accounts - Project Plan

## 1. Project Description
Konvy Accounts is a visually stunning digital account marketplace inspired by AE Empire Accounts. The site features a dark glassmorphism aesthetic with a Fortnite-focused marketplace vibe. This is a frontend-only visual rebuild with no backend functionality - all interactivity is simulated for design purposes.

## 2. Page Structure
- `/` - Home / Marketplace (hero, filters, product listing cards)
- `/login` - Login page with dark glassmorphism form
- `/register` - Registration page with dark glassmorphism form
- `/support` - Support page with contact/help sections

## 3. Core Features
- [ ] Dark glassmorphism design system
- [ ] Sticky navigation bar with logo and auth buttons
- [ ] Hero section with stats badges
- [ ] Announcement banner
- [ ] Search, filter, and sort controls
- [ ] Game category icon buttons
- [ ] Product listing grid with cards
- [ ] Login form
- [ ] Register form
- [ ] Support page
- [ ] Welcome tour modal (visual only)
- [ ] Responsive design for all screen sizes

## 4. Data Model Design
No database required - this is a visual-only build. Mock data will be hardcoded in components.

## 5. Backend / Third-party Integration Plan
None required - pure frontend visual build.

## 6. Development Phase Plan

### Phase 1: Design System & Home Page
- Goal: Set up dark theme, glassmorphism styles, fonts, and build the main marketplace page
- Deliverable: Fully styled home page with hero, filters, category buttons, and product listing cards

### Phase 2: Auth Pages
- Goal: Build Login and Register pages with matching glassmorphism forms
- Deliverable: Login and Register pages that match the design system

### Phase 3: Support Page & Polish
- Goal: Build Support page and add final responsive polish
- Deliverable: Complete 4-page site with consistent styling
